Useful Tips


Import HPSA Windows Patches:
This form will submit a job to OO that calls HPSA to run the /opt/opsware/mm_wordbot/util/populate-opsware-update-library script on the HPSA core. The job takes about an hour to run on a good day so sit back and wait for the HPSA job to email you the result. If an error is encountered please contact EAS; as something needs to be looked at on the HPSA core.
HPSA Patch Policy Migration Tool:
This form will submit a job to OO that calls HPSA to run the /opt/opsware/winpatch_promotion/patchpolpromote.sh script on the HPSA core. The job shold only about an half hour to run on a good day so sit back and wait for the HPSA job to email you the result. If an error is encountered please contact EAS; as something needs to be looked at on the HPSA core.
HPSA Content Migration Tool:
Example: You want to import the following software policies from:
/SAVVIS/Software Polices/Linux/RHEL Server 5/All/Application Engineering/OpenLogic-Apache_HTTPD/RHEL5_OpenLogic-Apache-HTTP-Worker-v2.2.22/
RHEL5_OpenLogic-Apache-HTTP-Worker-v2.2.22_x86_64_EN
RHEL5_OpenLogic-Apache-HTTP-PHP-Worker-v2.2.22_x86_64_EN
You will only add the base folder once. All content from the base folder will by synced.
